# PICKS Capstone Site

Static presentation site for the Project Inventory Compliance and Kitting System (PICKS) capstone.

## What is included

- Current information flow
- Option comparison and what-if scenarios
- System layout
- Information exchange examples
- Test cases
- Final materials
- Acronym reference

## Open locally

Open `index.html` in a browser.

No build step is required. The site uses plain HTML, CSS, and JavaScript.

## Key files

- `index.html` - home page
- `html/` - site pages
- `js/` - page interactions and data
- `styles.css` - shared styling
- `assets/` - logo, favicon, and final materials

## GitHub Pages

The site is intended to deploy from the repository root on the `main` branch.

The `.nojekyll` file is included so GitHub Pages serves the static files directly.
