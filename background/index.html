Context

Fragmented lab workflows make inventory, traceability, and compliance harder to manage.

PICKS is a systems engineering framework for improving how project inventory, lab records, enterprise data, and governance controls work together across GTRI lab environments.

Capstone scope

The project defines requirements, architecture, integration examples, verification thinking, and decision support. It is not a final production implementation.

Problem

Disconnected Tools

Labs use different tools and manual handoffs for procurement, receiving, inventory, lab execution, and compliance records.

Impact

Limited Visibility

Material status, part history, and project context can be difficult to trace across systems, especially when records live in separate places.

Need

Governed Access

Role-based access control and audit logging are needed so important records are protected and reviewable.

Direction

Enterprise Integration

The future ERP direction means PICKS should complement enterprise systems instead of duplicating ERP functionality.

Current State Physical View

How work moves today before the integrated architecture.

Request channels Email, tickets, conversations

Work starts with people and local lab practices.

Disconnected tools Workday, PEDYN, Sunflower, RFTrack, custom tools

Data is copied between systems instead of exchanged through one interface.

Tracking gaps Manual handoffs and direct delivery

Visibility can break after receiving, delivery, or lab-specific updates.

Project Story

How the Scope Evolved

1

Initial Need

Improve inventory visibility, lifecycle tracking, kitting, and compliance support for lab and project work.

2

Stakeholder Feedback

Lab interviews highlighted fragmented processes, inconsistent data entry, traceability gaps, and usability concerns.

3

Rescope

The USG ERP direction shifted PICKS toward integration readiness, future interoperability, and avoiding duplicate ERP functions.

4

Final Package

The final site, presentation, SEMP report, and MSOSA model provide a decision-support package for future PICKS planning.
